# Tutorial Notebooks

These notebooks mirror the tutorial workflows in an interactive format.

They use shared data under `examples/data` with paths like
`../../data/gx/tprim_2_4.out.csv`. Open or execute them with
`examples/tutorial/notebooks` as the working directory so those relative paths
resolve correctly.

Notebook checkpoints, untitled notebooks, generated PDFs, and duplicated data
files are intentionally not kept here.
