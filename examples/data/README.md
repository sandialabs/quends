# Example Data

This directory is the single shared data location for examples, tutorials,
notebooks, and notebook-driven tests.

- `cgyro/`: CGYRO CSV outputs at several collisionalities.
- `gx/`: GX single-run CSV outputs.
- `gx/ensemble/`: GX ensemble members used by the ensemble tutorials.
- `testdata/`: small synthetic or diagnostic datasets for workflow examples.

Keep tutorial code pointed at this tree rather than copying datasets into
notebook or script directories.
